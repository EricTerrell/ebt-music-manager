EBT Music Manager allows you to edit audio file metadata, and convert your audio files for playback on other devices. EBT Music Manager is open source.

github

EBT Music Manager uses the following components (and many more):

  • Electron
  • async-promise-pool
  • bootstrap-icons
  • electron-packager
  • ffmetadata
  • music-metadata
  • tabulator-tables

App icon by Everaldo Coelho

Transcoding and metadata manipulation performed by ffmpeg
